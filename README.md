# Detección y Organización de Citas Clínicas de Cáncer de Mama mediante Redes Neuronales Convolucionales

## 📖 Descripción

Este repositorio contiene el desarrollo de mi **Trabajo Fin de Máster (TFM)** del **Máster de Especialización en Inteligencia Artificial y Big Data**.

El proyecto tiene como objetivo desarrollar un sistema basado en **Redes Neuronales Convolucionales (CNN - Convolutional Neural Networks)** para la detección automática de información relevante en documentos clínicos relacionados con el **cáncer de mama**. Una vez identificada esta información, el sistema organiza y estructura las **citas clínicas**, facilitando su gestión y posterior análisis.

---

## 🎯 Objetivos

- Implementar un modelo de Redes Neuronales Convolucionales para la detección de información en documentos clínicos.
- Automatizar el procesamiento de documentación relacionada con pacientes con cáncer de mama.
- Extraer y organizar las citas clínicas de forma estructurada.
- Evaluar el rendimiento del modelo mediante diferentes métricas de aprendizaje automático.

## 📌 Objetivo final

El propósito de este trabajo es investigar la aplicación de técnicas de **Deep Learning** en el ámbito sanitario, desarrollando una solución capaz de detectar y organizar automáticamente información clínica relacionada con el seguimiento de pacientes con cáncer de mama, contribuyendo a optimizar la gestión de las citas clínicas.
